# solartesla
Charge your Tesla for free using excess solar

# Supported hardware

## Cars

* Tesla Model Y
* Tesla Model 3

## Solar inverters

* SolarEdge

# Config
Configure your Tesla and SolarEdge tokens in `config.py` starting from the sample:

```
cp config.py.sample config.py
```
